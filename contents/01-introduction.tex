\chapter{绪论}

\section{模板简介}
本模板（\texttt{muc-thesis}）旨在为中央民族大学（MUC）本科生提供一个符合学校最新排版规范、高度自动化且易于维护的 \LaTeX{} 毕业论文模板。

相较于传统的 Word 排版，本模板通过底层宏包配置，彻底解决了多级标题缩进、公式排版对齐、图表浮动编号、以及参考文献格式不统一等常见痛点。

写作者无需再将大量时间耗费在繁琐的格式微调上，可以全心专注于学术内容的创作。

\section{编译与排版环境}
为了正确渲染系统级中文字体（如宋体、黑体、方正小标宋等）并生成精准的交叉引用与目录，本模板强制依赖于 \textbf{XeLaTeX} 编译引擎。
\begin{itemize}
    \item \textbf{本地环境}：推荐使用完整版的 TeX Live 或 MiKTeX。标准编译链为 \texttt{xelatex} $\to$ \texttt{biber} $\to$ \texttt{xelatex} $\to$ \texttt{xelatex}。
    \item \textbf{在线环境}：若使用 Overleaf 等在线编辑器，请务必进入项目设置（Menu），将 \texttt{Compiler} 引擎切换为 \texttt{XeLaTeX}，随后点击 \texttt{Recompile} 即可由平台自动完成复杂的编译流程。
\end{itemize}

\textbf{表格技巧：}
\begin{itemize}
    \item \textbf{表题位置}：注意表格的 \texttt{\textbackslash caption} 必须写在 \texttt{tabular} 环境的上方。
    \item \textbf{列对齐}：在 \texttt{tabular} 参数中，\texttt{l} 代表左对齐，\texttt{c} 代表居中，\texttt{r} 代表右对齐。
    \item \textbf{自适应宽度}：如果表格内容极多，可以使用模板内置的 \texttt{tabularx} 环境实现自动换行。
\end{itemize}

\section{公式示例}
根据学校规范，公式应按章编号（例如式~\eqref{eq:quadratic}）。在 Word 中需要繁琐的 MathType 设置和 Tab 键对齐，而在 \LaTeX{} 中这一切都是自动完成的。

\begin{equation}
  x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
  \label{eq:quadratic}
\end{equation}

再如组合数公式~\eqref{eq:combination}：

\begin{equation}
  \frac{n!}{r!(n-r)!}
  \label{eq:combination}
\end{equation}

\textbf{使用要点：}
\begin{itemize}
    \item \textbf{环境选择}：请使用 \texttt{equation} 环境编写单行有编号公式。
    \item \textbf{自动编号}：无需手动输入编号，模板已配置为按章编号，格式为 (章-序号)。
    \item \textbf{字体一致性}：模板默认使用 Times New Roman 风格的数学字体，无需手动设置。
    \item \textbf{交叉引用}：使用 \texttt{\textbackslash label} 标记，\texttt{\textbackslash eqref} 引用。引用会自动带上圆括号，且编号会随顺序调整自动更新。
    \item \textbf{行内公式}：在正文中嵌入数学符号或简短公式时，请使用 \texttt{\detokenize{$...$}} 或 \texttt{\detokenize{\(...\)}} 将具体公式内容包裹（例如 \texttt{\detokenize{$E=mc^2$}}，渲染后即$E=mc^2$）。此类公式会随当前段落文本流排版，且不会分配独立编号。
    \item \textbf{小寄巧}：对于不熟悉Latex公式语法的同学，可以先在纸上手写需要的公式，然后使用带有\textbf{多模态理解}能力的AI产品（国产例如豆包/千问），让AI转化为Latex格式即可。
\end{itemize}
