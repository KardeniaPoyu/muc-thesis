\chapter{模板使用指南}

\section{必填项设置}
打开主文件 \texttt{main.tex}，在导言区必填项部分按注释提示修改以下命令：
\begin{itemize}
    \item \texttt{\textbackslash title\{...\}}：论文中文题目。
    \item \texttt{\textbackslash entitle\{...\}}：论文英文题目。
    \item \texttt{\textbackslash paperortype\{...\}}：设置为"论文"或"设计"，控制页眉显示。信计、统计专业填"论文"，光电、纳米专业填"设计"。
\end{itemize}

此外，导言区中有一行被注释的 \texttt{\textbackslash hideremarks} 命令，取消注释即可一键隐藏模板中所有红色提示信息（\texttt{\textbackslash remark\{\}}）。

\section{选填项：个人信息与前置页}
导言区中 \texttt{\textbackslash author}、\texttt{\textbackslash studentid}、\texttt{\textbackslash college}、\texttt{\textbackslash major}、\texttt{\textbackslash supervisor} 等个人信息为选填项，仅在需要用 \LaTeX{} 自动生成封面和前置页时填写。

\remark{推荐方案：使用 Word 模板填写好封面、任务书、承诺书、授权书并签字后，将这四页转为 PDF 存入 \texttt{frontmatter/} 文件夹，按 \texttt{main.tex} 中的注释说明插入即可。具体操作请参考 \texttt{main.tex} 中的详细注释。}

\section{章节与段落写作建议}

\subsection{如何增加新章节}
建议在 \texttt{contents/} 目录下新建 \texttt{.tex} 文件，例如 \texttt{05-new-chapter.tex}。然后在 \texttt{main.tex} 中使用 \texttt{\textbackslash include\{contents/05-new-chapter\}} 引入。

\subsection{段落写作规范}
\begin{enumerate}
    \item \textbf{首行缩进}：中文会首行自动缩进。你只需要正常撰写，换行时空一行即可开始新段落。
    \item \textbf{强调文本}：使用 \texttt{\textbackslash textbf\{粗体\}} 或 \texttt{\textbackslash textit\{斜体\}}。
    \item \textbf{列表环境}：
        \begin{itemize}
            \item \texttt{itemize} 用于无序列表（如本列表）。
            \item \texttt{enumerate} 用于有序列表（自动生成 1, 2, 3）。
        \end{itemize}
\end{enumerate}

\section{Overleaf 编译超时解决方法}
若在 Overleaf 中遇到编译超时（Timeout），可尝试以下方案：
\begin{enumerate}
    \item \textbf{局部编译}：在 \texttt{main.tex} 导言区使用 \texttt{\textbackslash includeonly\{contents/正在写的章节\}}。
    \item \textbf{草稿模式}：将 \texttt{\textbackslash documentclass} 改为 \texttt{\textbackslash documentclass[draft]\{muc-thesis\}}，可跳过图片渲染。
    \item \textbf{优化图片}：确保 \texttt{figures/} 下的图片单张大小不超过 1MB，推荐使用 PDF 或压缩后的 PNG。
\end{enumerate}
